OCY — OCR for Developers

Extract text from code screenshots, terminal output, and IDE windows. One line of code.

import ocy
result = ocy.extract_text("https://example.com/code.png")
print(result["text"])
const { extractText } = require("ocy");
const result = await extractText("https://example.com/code.png");
console.log(result.text);
String text = Ocy.extractTextOnly("https://example.com/code.png");
System.out.println(text);
auto result = ocy::extract("https://example.com/code.png");
cout << result.text << endl;

Quickstart

Install the SDK for your language and extract text in minutes.

Installation

pip install ocy

Usage

import ocy

result = ocy.extract_text("https://example.com/screenshot.png")
print(result["text"])
print(f"Confidence: {result['confidence']:.1%}")

API Reference

POST /api/extract

Extract text from an image URL.

Request

{
  "image_url": "https://example.com/screenshot.png"
}

Response

{
  "text": "extracted text...",
  "confidence": 0.95,
  "latency_ms": 150,
  "model": "ocy-v1-int8",
  "chars_detected": 42
}
FieldTypeDescription
textstringExtracted text
confidencefloatConfidence score 0.0-1.0
latency_msintProcessing time in ms
modelstringModel identifier
chars_detectedintCharacter count
GET /api/health

Check API health status.

{
  "status": "ok",
  "version": "1.0.0",
  "model": "ocy-v1-int8"
}

Error Codes

CodeDescription
400Invalid request (missing image_url, invalid URL format)
429Rate limit exceeded (100 requests/hour)
502Failed to fetch image
500Internal inference error

Self-hosting

Run your own OCY worker with Cloudflare Workers.

  1. Clone the repository

    git clone https://github.com/solez-ai/ocy.git
cd ocy

  2. Deploy to Vercel

    npm install -g vercel
vercel

    Or connect your repo to Vercel for automatic deployments.
